# LocalAssistAI

LocalAssistAI is an AI-powered customer support automation system that runs entirely on local LLMs.

## Features
- Data masking (PII protection)
- Sentiment analysis
- Ticket classification (category + priority)
- Auto-response generation
- Escalation logic
- Fully offline (no external APIs)

## Tech Stack
- FastAPI
- SQLite
- Ollama (Local LLM)
